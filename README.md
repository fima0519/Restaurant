# Restaurant Management System (Java)

A comprehensive "Java-based Restaurant Management System" that manages employees, customers, food inventory, orders, and receipts. This console-based application demonstrates object-oriented programming (OOP) principles and real-world application development in Java.

---

Technologies Used:
- Java – Core programming language for logic and functionality  
- Object-Oriented Programming (OOP) – Classes, inheritance, encapsulation, and polymorphism  
- IDE – NetBeans / IntelliJ IDEA / Eclipse

---

Functional Features:

Person
- Maintain basic information such as ID, name, and contact details.

Employee
- Store and manage employee details (job title, salary).  
- Update employee job titles and salaries.

Customer
- Maintain a list of orders per customer.  
- Place new orders, view order history, and check total spending.

EmployeeManager
- Add or remove employees.  
- Find employees by ID or job title.  
- Show all employees.  
- Find employees with the highest salary.

CustomerManager
- Add or remove customers.  
- Find customers by ID.  
- Update customer contact info.  
- Provide total number of customers.

Food & Inventory
- Store food information: ID, name, price, quantity, cuisine, type, and description.  
- Update price and quantity.  
- Add or remove food items from inventory.  
- Search food by ID, cuisine, type, or price range.  
- Check food availability.

Menu & MenuPart
- Organize menu sections: appetizers, mains, desserts.  
- Add or remove food items in menu sections.  
- Display menu items and availability.  
- Search menu items by cuisine or type.

Order & OrderManager
- Maintain order details: order ID, customer ID, date, payment, status, and list of food items.  
- Add or remove food items from orders.  
- Calculate total payment.  
- Add, remove, or update orders.  
- Retrieve orders by customer, status, or date range.

Receipt
- Generate receipts after payment.  
- Include receipt ID, order ID, customer ID, payment details, total amount, and date.
